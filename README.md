
# Personal Finance Management CLI

A Python-based command-line application for managing personal finances.  
The system allows users to track expenses, set monthly budgets with alerts, and monitor savings using CSV files for persistent storage.

---

## Features

- User registration and login
- Expense tracking with categories
- Monthly budget management with usage alerts (50%, 80%, exceeded)
- Savings tracking
- Automatic CSV file initialization
- Simple and beginner-friendly CLI interface

---

## Tech Stack

- Python 3
- CSV file storage
- Standard Libraries: `csv`, `os`, `re`, `datetime`

---

## How It Works

- Required CSV files are automatically created on first run
- User data and financial records are stored persistently
- Budget alerts are triggered based on usage percentage
